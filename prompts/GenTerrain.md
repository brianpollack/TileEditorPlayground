## Your Job

I am giving you an image as a reference that you should use to understand
how tiles in a game are setup.  This image contains and 8 x 8 tile grid
where each tile is 128x128 pixels in size.

In my image, Green is grass, Brown is dirt.

Your job is to re-imagine the tiles using these style guides.  Do not add
any new elements.  Do not add trees, bushes, or other objects.   I'm
giving you specific elements for Grass and Dirt and I need you to only
use Grass and Dirt.

## Design

STYLE: Stylized Hand-Painted Farming / Exploration RPG Asset Style

MUST use a base asset size of 128x128 pixels for this 8x8 grid.
DO NOT including lighting, shading or unique aspects as we are creating re-usable tiles.
MUST use a hand painted style with clean color sets but not photo realistic
MUST use a fantasy faire world style
SHOULD Smooth locations where grass meats dirt
MUST add random pebbles and weeks and things within the tiles to add some character

MUST use hand-painted digital texture style
MUST maintain smooth painterly surfaces
SHOULD keep texture noise low
DO NOT use photorealistic materials
DO NOT use gritty or noisy texture patterns
MUST keep colors vibrant but slightly softened
MUST maintain consistent palette saturation across assets
SHOULD use earthy greens and warm browns for environment assets
SHOULD use warm accents for buildings and props
DO NOT use neon colors
DO NOT use overly dark desaturated tones
